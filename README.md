### Запуск приложения на локальном хосте

Клонировать проект

Для запуска приложения на локальной машине выполнить команду в директории проекта
```commandline
python manage.py runserver
```
После чего по появившейся ссылке можно открыть проект:
http://127.0.0.1:8000/
